## Decisions Made

- **Version selection:** Used `0.2.90` per PRODUCT.md §16 + REQUIREMENTS.md SEC-01 wording. Although `0.2.117` is the current `latest` dist-tag, `0.2.90` is the lowest published GHSA-5474-4w2j-mq4c-fixed release. Choosing the lowest fixed version minimizes unrelated SDK type-surface churn that could otherwise complicate downstream SEC-02..SEC-08 plans this wave gates.
- **Exact pin (no caret):** Required by SEC-01's "pinned to a fixed version" wording. Caret would re-admit any vulnerable patch released later in `0.2.x`.
- **Left `overrides["@anthropic-ai/sdk"]: "^0.86.1"` untouched** per plan note 5 — it is an intentional transitive-conflict resolution.
- **Did not opportunistically bump `zod` / `commander` / dev-deps** — out of scope for SEC-01.
